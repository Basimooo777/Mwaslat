First Step

click on 'Register'

Second Step

-Enter email in specified field then
-Enter your password then
-Retype your password
-Then click on 'signup'
-Then you should activate your account through a message sent to your email

Third Step

Go to home page then click on 'login'

Fourth Step

-Once after your account is activated
-Enter your email and password
-You can check the 'Remember me' button to make your browser save your login data so you don't need to write it every time you open our system
-Then Click on 'Sign in' button

Fifth Step

-A Message would show that you are singed in
-And a 'logout' link so you can log out of our system

First Step

-Click on 'Go To' then
-Choose 'My Nodes' from the list and click it

Second Step

-It will show you table with all your added nodes and the table would be empty if you didnt add any node before
-Click on the 'New' button to add new node

Third Step

-Enter name of Node (District) to be added
-Click on 'Add Place' to locate its position on map

Fourth Step

-After adding Node name and position on map then
-Click on 'Save' button

Fifth Step

-Now you can see list of your nodes with our new added node
-Each node can have 3 actions :
1)green button to show node on map
2)blue button to edit node data
3)red button to delete node from system

First Step

-Click on 'Go To' then
-Choose 'My Routes' from the list and click it

Second Step

-It will show you table with all your added routes and the table would be empty if you didn't add any route before
-Click on the 'New' button to add new route

Third Step

-Zoom in the map to see the districts to be connected

Fourth Step

--right click on the specified district then choose 'select'

Fifth Step

-You will see name of the node added
-Then you can either add many other stop by the same way as before Or by adding new node which didnt exist by clicking 'add place' on top right corner of map

Sixth Step

-Now specify the transportation type , Cost and Way description if it exist
-Then click on 'Create Route'

Seventh Step

-You will see your route been added to the table
-the 3 action buttons (blue button to edit it ,red button to delete it ,green button to show it)
